# Message Template

Use this format in `agent-communication-log.md`.

```md
## YYYY-MM-DD HH:MM - Agent Name - short status

Status: starting / running / blocked / handoff / complete

Task:

Files likely to change:

Files claimed:

FACT:

INTERPRETATION:

HYPOTHESIS:

GAP:

Checks:

Next:
```

Keep entries concise. Use repo-relative paths only.
