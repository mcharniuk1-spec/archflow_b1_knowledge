# Output Documents

This folder defines the documents produced by the Block 1 workflow.

Current status: templates exist, but the first real proof run has not been completed yet.

## Expected Run Output Set

Each full run should produce:

- `source-inventory.md`
- `retrieval-log.md`
- `context-digest.md`
- `prd.md`
- `responsibility-matrix.md`
- `knowledge-base-update.md`
- `review-report.md`
- `run-summary.md`

## Rule

Do not store raw private transcripts, private board exports, credentials, local paths, or private links in output documents.
