# Memory Architecture

## Authority Order

1. Project plan and public WikiLLM.
2. Run folders and review reports.
3. LlamaIndex retrieval over approved corpus.
4. Future Cognee operational recall.
5. Future vector indexes such as turbovec.

## Promotion Path

`source -> inventory -> digest -> draft output -> review -> memory candidates -> WikiLLM -> retrieval rebuild -> optional Cognee remember -> readback`

## Cognee Rule

Cognee must not replace WikiLLM. It can only remember approved summaries, PRDs, decisions, research cards, and source registry facts after AF Review.

## turbovec Rule

turbovec is a local retrieval acceleration layer under LlamaIndex. It is not a source of truth and should store no public repo artifact by default.
