# Legacy File Inventory

This is a sanitized inventory of legacy file categories. It is not a raw copy of legacy files.

| Category | Public Handling |
|---|---|
| Static website HTML/CSS/JS | Not copied. Review separately before public release. |
| API route files | Not copied. May contain deployment assumptions or private integration details. |
| Deployment metadata | Not copied. May contain project/account metadata. |
| Screenshots and QA images | Not copied. May contain visual state, metadata, or private context. |
| Source PDFs and office documents | Not copied. May contain private authorship, metadata, or unpublished strategy. |
| Workflow reports | Summarized only. Raw reports may contain local paths or private source references. |
| Browser automation logs | Not copied. May contain local runtime or page state. |
| Figma sync artifacts | Not copied. May contain private workspace references. |

To make any legacy file public, run a separate public-safety review first.
