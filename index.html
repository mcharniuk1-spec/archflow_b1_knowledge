How Work Moves

The process is designed for traceability. It separates confirmed facts from interpretation, hypotheses, and gaps before anything becomes a public claim.

  1. 01

    Context Intake

    Collect approved notes, calls, tickets, docs, research, and constraints.

  2. 02

    Evidence Model

    Classify sources, jobs, stakeholders, alternatives, risks, and open questions.

  3. 03

    PRD And Tasks

    Build the PRD, acceptance criteria, responsibility matrix, and task sequence.

  4. 04

    Control Review

    Run safety, evidence, memory, and implementation gates before handoff.
