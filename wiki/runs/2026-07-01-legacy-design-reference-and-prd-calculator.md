# 2026-07-01 Legacy Design Reference And PRD Calculator

## Task

Use the latest legacy ArchFlow website as the design reference for the future public PRD/ICP landing page.

## Result

Created `project/reports/2026-07-01-legacy-design-reference-and-prd-calculator-brief.md`.

## Key Finding

The legacy website should be reused for visual mechanics, not category positioning. Its strongest reusable elements are layered 3D arch assets, scroll drift, cursor hover-depth, warm editorial styling, dark command panels, and a calculator-first proof interaction.

## Recommendation

The new public website should simplify the legacy ROI/readiness calculator into a PRD/ICP Operating ROI Calculator with two modes:

- PRD/ICP ROI;
- Knowledgebase Readiness.

The calculator should estimate recoverable product-work value, knowledgebase readiness, and the recommended first ArchFlow package while avoiding guaranteed ROI claims.

## Confidence

Confidence: 0.82 for visual direction; 0.68 for calculator economics until ROI and demand proof exist.
